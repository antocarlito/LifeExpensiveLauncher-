2. Fonctionnalites

Launcher (C# WPF .NET 10)

  • Page d'accueil animee (splash)
  • Verification et telechargement automatique des mods
  • Scan anti-triche (PBO + fichiers) avec token signe HMAC-SHA256
  • Re-scan au clic JOUER (detection modifications post-scan)
  • Statut serveur en temps reel (joueurs en ligne)
  • Playlist musique configurable depuis le serveur
  • Image/video de fond
  • Mode maintenance avec bandeau + popup personnalisee
  • Mode whitelist avec lien vers inscription
  • Liens rapides : TeamSpeak, Discord, Site Web, TFR
  • Changelog des mises a jour
  • Lancement via arma3battleye.exe (BattlEye actif)
  • Journal selectionnable/copiable
  • Parametres en overlay
  • Volume slider + nom musique en cours
  • Fenetre redimensionnable

3. Securite anti-triche

CoucheProtectionQuand
LauncherScan PBOs + hash + token HMAC-SHA256Demarrage + clic JOUER
API PHPSignature verifiee + rate limiting (10 req/min)A chaque token
SQF serveurVerifie token en DB, kick si absentConnexion joueur
CfgRemoteExecWhitelist mode=1Permanent
BattlEyeProtection memoire/injectionPermanent

Flux du token

1. Launcher scanne les mods du joueur → hash global

2. Signe avec HMAC-SHA256 (secret partage) + timestamp

3. Envoie au serveur PHP → verifie signature → stocke en DB (valide 15 min)

4. Clic JOUER → re-scan + nouveau token frais

5. Joueur se connecte → SQF verifie le token en DB → OK ou kick

5. Configuration

launcher_config.json (fichier unique)

Toute la configuration se fait dans ce fichier sur le serveur boot.

Adresses serveur, liens TS/Discord, URLs mods/musiques, etc.

Le launcher le telecharge au demarrage et s'adapte automatiquement.

6. Build et deploiement

Compiler le launcher

dotnet publish LifeExpensiveLauncher -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o publish

Creer l'installateur

build_installer.bat

Resultat : installer/LifeExpensive_Launcher_Setup.exe

Signer l'exe (Certum SimplySign)

signtool sign /fd sha256 /tr http://timestamp.certum.pl /td sha256 /csp "SimplySign CSP" /n "antocarlito" LifeExpensiveLauncher.exe
